# Retold SynthDatabeacon

> An on-demand synthetic-record source: a meadow-shape REST surface with no database behind it, generated deterministically from declarative specs

- Serve realistic, repeatable records from a JSON spec
- Deterministic per-cell seeding — same spec and seed, same data every time
- A drop-in MeadowProxy source for retold-databeacon over the Ultravisor mesh
- Bundled industrial-supply (14 entities) and customers-demo (2 entities) specs
